# Frontend Shared Resources

Deze map groepeert alle herbruikbare frontendresources, zoals componenten, styles en utility-functies. Organiseer submappen per domein (`components/`, `styles/`, `utils/`) om hergebruik tussen Vue, React of toekomstige frameworks eenvoudig te houden.

## Werkwijze

1. Hou de code framework-onafhankelijk waar mogelijk.
2. Publiceer resources via npm workspaces of gebruik path-aliasen.
3. Documenteer breaking changes in een changelog of in de component README.
